# [Ngen Django Backend](https://github.com/CERTUNLP/ngen)
## ✨ Quick Start in `Docker`

> Get the code

```bash
$ git clone https://github.com/CERTUNLP/ngen.git
$ cd ngen
```

> Start the app in Docker

```bash
$ cd docker
$ docker-compose up --build 
```

> API server on port `8000`

## ✨ Default credentials
```javascript
ngen:ngen
```
